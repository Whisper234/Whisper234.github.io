2021-11-3 ～ 2021-11-5

1. 学习git并在电脑上操作

2. 一些命令

  • git init 初始化仓库
  • git add 添加文件到暂存区
  • git commit 将暂存区内容添加到仓库中
  • git status 查看仓库当前状态，显示有变更的文件
  • git diff 比较文件的不同，即暂存区和工作区的差异
  • git reset 回退版本
  • git log 查看历史提交记录
  • git checkout 切换分支命令
  • git merge 合并分支命令
  • git branch 列出分支基本命令
  • git pull 命令用于从远程获取代码并合并本地的版本
  • git push 命令用于从将本地的分支版本上传到远程并合并
  • git tag 标签

二、遇到的问题

  1. git的语句还是比较多的，很容易记混和记不住
  2. 反复尝试才得到把写的文件上传上去
  3. markdown语法记不清楚
  4. 不知道这篇博客写什么内容

三、解决问题

  • 1.2. 3. 多敲一敲
  • 4.硬着头皮写
