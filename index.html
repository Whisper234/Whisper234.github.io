1.9清除内外边距

网页元素很多带有默认的内外边距，所以要清除。

行内元素尽量只设置左右内外边距，不设置上下边距，但转为块和行内块就可以。

  • 怎么去除li前面项目符号：

圆角边框

boder-radius属性用于设置元素的外边框圆角。

语法：

  • 参数值可以用px也可以用百分比。
  • 圆角矩形设置为高度的一半
  • 可以跟四个值，左上，右上，右下，左下。

盒子阴影

box-shadow属性添加阴影。

语法：

值 描述
h-shadow 必需，水平阴影的位置，允许负值
v-shadow 必需，垂直阴影的位置，允许负值
blur 可选，模糊距离
spread 可选，阴影尺寸
color 可选，阴影颜色
inset 可选，将外部阴影改为内部阴影

阴影不占空间，不影响其他的大小。

文字阴影

text-shadow属性将应用于文字阴影。

语法

值 描述
h-shadow 必需，水平阴影的位置，允许负值
v-shadow 必需，垂直阴影的位置，允许负值
blur 可选，模糊距离
color 可选，阴影颜色

1. 学习git并在电脑上操作

2. 一些命令

  • git init 初始化仓库
  • git add 添加文件到暂存区
  • git commit 将暂存区内容添加到仓库中
  • git status 查看仓库当前状态，显示有变更的文件
  • git diff 比较文件的不同，即暂存区和工作区的差异
  • git reset 回退版本
  • git log 查看历史提交记录
  • git checkout 切换分支命令
  • git merge 合并分支命令
  • git branch 列出分支基本命令
  • git pull 命令用于从远程获取代码并合并本地的版本
  • git push 命令用于从将本地的分支版本上传到远程并合并
  • git tag 标签
